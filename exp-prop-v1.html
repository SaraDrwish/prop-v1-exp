Pinhole geometry & spatial formulas

Using stable bounding box width w (pixels) and center xc from attention‑based detector (Hossam 2026).

📏 Distance Z = (Wreal × F) / w
↔️ Lateral offset X = ((xc - cx) × Z) / F
⬆️ Height Y = Z × sin(θtilt)

📌 Parameters fixed (calibration/assumption):
Wreal = 0.4 m (average drone wingspan), F = 1000 px (focal length), cx = 320 px (optical center, 640px image width)
θtilt = 15° (camera downward tilt)  →  sin(15°) ≈ 0.2588

Why attention‑stabilized box? Standard YOLO jitter (2–3 px) can cause distance errors >15%. Hossam’s CBAM+ECA reduces bounding box noise → more reliable X,Y,Z.

Real‑world scenario: Power tower protection

Camera mounted on tower → live feed → our model outputs (X, Y, Z) for every detected drone.
➤ X : lateral offset (left/right of tower)
➤ Z : distance from tower → if decreasing → approach alert
➤ Y : height relative to ground → collision risk assessment
🚨 Decision logic: When Z < 30m and |X| < 5m and Y within tower height → Red alert (active countermeasure).
